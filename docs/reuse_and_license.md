# Reuse And License Notes

This project was created after studying n8n, LangGraph, CrewAI, and public n8n workflow collections.

## What Was Reused

- Architectural ideas.
- Publicly documented concepts.
- Project organization patterns.
- Workflow design patterns.

## What Was Not Reused

- No source code copied from n8n, LangGraph, CrewAI, or n8n-workflows.

- No proprietary or enterprise files copied.
- No credentials or external secrets copied.
